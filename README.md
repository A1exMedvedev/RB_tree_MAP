# RBTreeMap: Реализация Красно-Чёрного Дерева на Go с Generics

![Go CI Tests](https://github.com/A1exMedvedev/RB_tree_MAP/actions/workflows/go.yaml/badge.svg)

`RBTreeMap` — это типобезопасная реализация **Красно-Чёрного Дерева** на языке Go, использующая дженерики (Generics), появившиеся в Go 1.18. Эта структура данных предоставляет упорядоченную map/dictionary с гарантированной логарифмической сложностью O(log N) для операций вставки, удаления и поиска.

Красно-чёрное дерево — это вид самобалансирующегося двоичного дерева поиска, которое идеально подходит для задач, требующих отсортированных данных и быстрой производительности.

## Особенности

-   **Типобезопасность**: Полная проверка типов во время компиляции благодаря дженерикам.
-   **Высокая производительность**: Гарантированная сложность O(log N) для ключевых операций.
-   **Минимальные аллокации**: Оптимизировано для низкого потребления памяти.
-   **Стандартный интерфейс**: Методы, аналогичные стандартным коллекциям (`Get`, `Insert`, `Remove`).
-   **Итераторы**: Удобный обход дерева в отсортированном порядке с помощью `iter.Seq2`.
-   **Кастомные компараторы**: Возможность задать свою функцию сравнения ключей.
-   **Надежность**: Код полностью покрыт unit-тестами, тестами на поведение и бенчмарками.
